# Nibu Solutions

A responsive marketing website for a fictional software company, built as a portfolio project. It showcases services, how the team works, client testimonials, and a contact form.

## Tech Stack

| Library | Version |
|---|---|
| React | 19 |
| TypeScript | 5.9 |
| Vite | 7 |
| Tailwind CSS | 4 |
| Framer Motion | 12 |
| React Router DOM | 7 |
| React Hook Form | 7 |
| Zod | 4 |
| shadcn/ui + Radix UI | — |
| Lottie React | 2 |
| Embla Carousel | 8 |
| Lucide React | 0.562 |

## Getting Started

**Requirements:** Node.js 18+

```bash
# Clone the repository
git clone https://github.com/lgimenez-dev/nibu-software-site.git
cd nibu-solutions-website

# Install dependencies
npm install

# Start the dev server
npm run dev
```

The app will be available at `http://localhost:3000`.

```bash
# Build for production
npm run build

# Preview the production build
npm run preview
```
